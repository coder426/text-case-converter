== Description ==

Text Case Converter is a open source wordpresss plugin using which you can change your page or post text into uppercase or lowercase or capitalize or sentence case or invert case etc.. Besides this, you can also generate dummy text in your post and can count the no. of characters, words and lines in post text.

Watch this video to see how the plugin works:

[youtube https://www.youtube.com/watch?v=YMtaGXHyA70]

= Key features of this plugin include =

* Transform your Page / Post Text in uppercase / lowercase / capitalize.
* Generate Dummy Text within Tinymce editor.
* Calculate no. of characters, words and paragraphs.
* Superscript / subscript your text.
* Copy to Clipboard and paste it anywhere.
* Download your post / page text in separate file.
* Add Break row to your text.
* Clean all text. 
* Highlight text.
* Quotation marks around the.
* HTML5 tag for addresses.
* Abbreviation tag with title attribute for text.
* Tooltip and synonyms features for text.
* New buttons for audio and video insert.
* Compatible with Classic Block in Gutenberg Block Editor.


== Installation ==

The plugin is simple to install:

 * Download text-case-converter.1.0.zip
 * Unzip
 * Upload catcbll directory to your /wp-content/plugins directory
 * Go to the plugin menu page and activate the plugin.

== Frequently asked questions ==

= How to use Text Case Converter plugin? =

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Then go to new post or page and enable classic editor, you will see extra buttons to use new features.


= What is the benefit of using this plugin? =

User can easily change page or post text into uppercase or lowercase or capitalize or sentence case or invert case etc..
